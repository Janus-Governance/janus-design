# Janus Design

Minimal visual baseline for Janus publication surfaces.

## Scope

This repository defines the shared visual baseline for Janus web surfaces.

Included:
- design tokens
- typography baseline
- link interaction baseline
- shared container rhythm

Excluded:
- surface-specific CTA systems
- card systems
- framework hero buttons
- apex-only visual patterns

## Files

- `janus-tokens.css` → canonical shared tokens
- `janus-base.css` → minimal reusable baseline

## Adoption model

Recommended order:
1. core
2. docs
3. apex
4. framework (only if/when intentionally migrated)

## Principle

Stabilize first.
Abstract second.
